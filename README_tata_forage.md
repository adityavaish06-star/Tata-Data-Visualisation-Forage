## Overview

Virtual experience programme simulating real data analytics tasks for Tata
Group's executive leadership team. Completed all 4 tasks in March 2026.

---

## Tasks Completed

### Task 1 — Framing the Business Scenario
- Identified the right analytical questions before touching any data
- Understood what the CEO and CMO actually needed vs. what the data showed
- Developed a question framework for executive-level analysis

### Task 2 — Choosing the Right Visuals
- Evaluated which chart types communicate different business insights clearly
- Avoided common visualisation mistakes (misleading axes, wrong chart types)
- Selected appropriate visuals for time-series, categorical, and part-to-whole data

### Task 3 — Creating Effective Visualisations
- Built KPI dashboards in Power BI monitoring cross-functional performance
- Transformed multi-source datasets into clean, executive-ready visual reports
- Applied best practices for colour, layout, and data labelling

### Task 4 — Communicating Insights
- Presented findings framed around business impact, not just data
- Structured narrative: context → insight → recommendation
- Delivered a boardroom-ready summary for Tata's senior leadership

## Key Learning

How to translate raw business data into boardroom-ready insights.
Choosing the right chart type is a business decision, not a design one —
the most important question is always: *what does leadership need to act on?*
